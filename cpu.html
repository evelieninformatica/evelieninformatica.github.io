Basis CPU

CPU, wat betekend dat nou? De letterlijke betekenis van CPU is Central Processing Unit. Dit wordt ook wel de processor genoemt.
De processor is het hart van je computer. Het bestuurt je hele apparaat. De meeste appraraten hebben één processor, maar er zijn computers met meerdere,
dit wordt multicoreprocessor genoemt.

Multicoreprocessor

Als een apparaat meerdere processoren heeft wordt dat een multicorprocessor genoemt. de processor heeft meerdere kernen,
die apart van elkaar informatie kunnen vewerken. Hierdoor kan een computer meerdere taken doen tegelijkertijd, multitasking.
Hier door gaat het verwerken veel sneller dan wanneer een processor maar een kern heeft. Processoren worden aangeduid in cores.
Voor de lagere aantallen worden daarvoor verschillende termen gebruikt.

  • Singlecore (1 kern)
  • Dualcore (2 kern)
  • Quadcore (4 kern)
  • Hexacore (6 kern)
  • Octacore (8 kern)

    • Taken van CPU

    Het CPU kun je verdelen in:

  • Een besturingsorgaan
  • Een rekenorgaan


    • De taken van een processor zijn:

  • Het ophalen van instructies en gegevens uit het interne geheugen
  • Het laten uitvoeren van bewegingen door het rekenorgaan
  • Het opslaan van resultaten in het computergeheugen
  • Het regelen van de uitvoer van gegevens, bijvoorbeeld naar beeldscherm of printer

    • Snelheid van een processor

    De snelheid van de processor heeft te maken met de versnellingssnelheid.
    Dit geeft de hoeveelheid instructies aan die de processor in één seconde kan verwerken.
    Dit wordt aangegeven in mips. Dat staat voor
    millions of instructions per second.
    Het kan ook aangegeven worden in de kloksnelheid. Deze wordt bepaald door de klokchip. Die
    heeft een vast aantal malen per seconde en geeft een soort puls aan de processor om alle verweringsstappen
    gelijk te laten lopen. Dit drukken we uit in hertz.

    Cachegeheugen

    Het cachegeheugen zorgt ervoor dat de processor nog sneller kan werken.
    het ophalen van instructies en gegevens uit het interne geheugen kost tijd. Deze
    tijd kan versneld worden door instructies en gegevens op te slaan in extra snel werkgeheugen,
    dat de processor rechtsstreeks kan gebruiken en dat dicht bij die processor zit.
    Dit type geheugen wordt cache genoemd.
